# YUCLAW Financial Intelligence Plugin for OpenClaw

The financial brain OpenClaw needs but cannot build.

## What this gives OpenClaw users

- `/yuclaw regime` — Live CRISIS/RISK_OFF/RISK_ON detection
- `/yuclaw signals` — Real factor-scored buy/sell signals
- `/yuclaw brief` — Nemotron 120B institutional brief
- `/yuclaw risk` — Real VaR/CVaR/Kelly calculations
- `/yuclaw zkp` — Latest on-chain cryptographic proof

## Why YUCLAW is different

| Feature | OpenClaw | YUCLAW |
|---|---|---|
| Financial backtests | No | Calmar 3.055 |
| ZKP audit trail | No | On-chain |
| Real VaR | No | Historical sim |
| Local 120B model | No | Nemotron DGX |
| Verified track record | No | Building daily |

## Install
```bash
openclaw plugins install yuclaw-financial
```
